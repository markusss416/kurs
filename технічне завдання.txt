ТЕХНІЧНЕ ЗАВДАННЯ ДЛЯ ПРОЕКТА:
ІНФОРМАЦІЙНА СИСТЕМА управління рейсами літаків
Дана інформаційна система має містити дані про клієнтів(ім’я, прізвище,телефон).
Також ця система має містити інформацію про польоти,напрямок польотів,дату).
Типовими для ІС є запити:
Запит на вибір всіх стовпців з таблиці users для перевірки чи існує користувач з таким логіном
Запит на видалення інформації з таблиці flight
Запит на поновлення статусу польоту
Запит на вибір всіх стовпців з таблиці fligt_type

Розширене завдання
Запит SQL з використанням JOIN для об'єднання таблиць flight і flight_type для перевірки статусу заявки для користувача
Запит SQL з використанням JOIN для об'єднання таблиць flight і flight_type для перевірки статусу заявки в адміна
